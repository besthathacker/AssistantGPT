# AssistantGPT Full iOS App Project

This Xcode project integrates AssistantGPT with AppIcon and Splash folders.

## What's Included
- **AssistantGPT.xcworkspace** – Open this in Xcode.
- **Resources/index.html** – Main AssistantGPT interface.
- **Resources/AppIcon/** – App icons (replace placeholders).
- **Resources/Splash/** – Splash screen images (replace placeholders).

## How to Build
1. Open `AssistantGPT.xcworkspace` in Xcode.
2. Set your signing & provisioning profile in **Signing & Capabilities**.
3. Replace placeholder AppIcon and Splash files with real assets.
4. Build and run on your iOS device or simulator.

Enjoy using AssistantGPT!
