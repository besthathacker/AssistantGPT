# AssistantGPT Xcode Project

This is a preconfigured Xcode project for the AssistantGPT app.

## Instructions
1. Open `AssistantGPT.xcworkspace` in Xcode.
2. Add signing certificates and provisioning profiles in Xcode preferences.
3. Replace the placeholder `index.html` with the full AssistantGPT HTML file.
4. Build and run!

This project uses SwiftUI and a WebView to load `index.html`.
